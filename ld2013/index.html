LinuxDay 2013

OpenStack

as GNU/Linux

the Project

A cloud operating system that controls large pools of compute, storage, and networking resources

An IaaS platform, initially built to displace/replace Amazon AWS

10,000 feet view

A collection of independent software packages

Distributed under Apache 2.0 license

Together offer a certain level of compatibility with the Amazon API but want to be more than that

Stackalytics (havana)

~127k lines of code (Python/Bash)

~1k developers

Releases a "major" every 6 months

Amongst the contributors also Rackspace, Red Hat, HP, Nebula, IBM, Mirantis, VMware, ...
